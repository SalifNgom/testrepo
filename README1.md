# Heart Disease Prediction Project

## Project Overview
This project aims to predict heart disease using machine learning techniques based on clinical data.

## Objectives
- Explore the dataset
- Prepare the data
- Train a classification model
- Evaluate model performance
- Interpret the results

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Results
The model achieved:
- Accuracy: 83%
- Precision: 83%
- Recall: 80%

## Conclusion
The model shows promising predictive performance for identifying patients at risk of heart disease. However, reducing false negatives remains essential for medical applications.
